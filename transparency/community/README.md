# Community Feature Request Transparency

This directory documents what data is stored for feature requests and moderation.

## Included artifacts

- `schema.sql` - canonical public schema for feature requests
- `DATA_POLICY.md` - plain-language data policy

## Scope

This covers feature request data handling only. Product-layer transport,
credentials, and runtime operations remain private.

## Sync policy

If the product schema or data-handling logic changes, this directory must be updated in the same release cycle.
