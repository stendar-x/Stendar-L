# Stendar-L Protocol

Public source for the Stendar lending protocol on Solana.

This repository contains:

- `programs/stendar/` - Anchor on-chain program
- `sdk/` - `@stendar-x/sdk` TypeScript SDK (published to npm)
- `transparency/` - inspectable platform-stats and community-data methodology
- `security/` - collateral listing manifests and schema
- `tests/` - protocol TypeScript regression tests

## Quick Start

```bash
npm ci
npm --prefix sdk ci
npm run test:ci
npm --prefix sdk test
cargo test --manifest-path programs/stendar/Cargo.toml
```

`npm run test:ci` runs the root local CI gate: security script tests, deterministic
pending-test scan, regression suite, artifact-aware IDL sync check, and audit gate.

## IDL Workflow

The canonical IDL is generated by Anchor at `target/idl/stendar.json`.

Run the strict sync check after generating artifacts (for example, after
`anchor build`):

```bash
npm run idl:check
```

This verifies `target/idl/stendar.json` is in sync with `sdk/src/idl/stendar.json`.

For artifact-optional environments, run:

```bash
npm run idl:check:if-present
```

This mode passes when `target/idl/stendar.json` is absent and validates sync when it
is present.

## Anchor Defaults

`Anchor.toml` defaults the provider to `devnet` and uses wallet
`~/.config/solana/id.json`.
