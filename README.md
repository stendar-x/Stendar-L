# Stendar-L Protocol

Public source for the Stendar lending protocol on Solana.

This repository contains:

- `programs/stendar/` - Anchor on-chain program
- `sdk/` - `@stendar/sdk` TypeScript SDK (published to npm)
- `transparency/` - inspectable platform-stats and community-data methodology
- `security/` - collateral listing manifests and schema
- `tests/` - protocol TypeScript regression tests

## Quick Start

```bash
npm ci
npm run test:regression
npm --prefix sdk ci
npm --prefix sdk test
```

## IDL Workflow

The canonical IDL is generated by Anchor at `target/idl/stendar.json`.

Run:

```bash
npm run idl:check
```

This verifies `target/idl/stendar.json` is in sync with `sdk/src/idl/stendar.json`.

## Publishing the SDK

```bash
npm --prefix sdk version patch
npm --prefix sdk publish
```

## Transparency

- `transparency/platform-stats/` documents and publishes the exact stats-calculation logic used by the product.
- `transparency/community/` documents feature-request data handling and published schema.
